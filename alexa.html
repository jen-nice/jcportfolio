This past November I took part in The Lady Hacks - the first all women hackathon in Canada! My team and I made a personal reminder assistant using the Amazon Echo, Google Script, chrome extensions and IFTTT technology. Using Alexa (on the Amazon Echo) the user can add an event to their Google Calendar and then when the user runs the script reminders are generated for each event in the calendar. Now when it is time for an event, Alexa will speak up and let the user know. I was responsible for the Google script development, configuring the Amazon Echo and experimenting with other hardware such as the Raspberry Pi and Intel Edison to see which hardware would work best. The project was inspired by a fellow team member who wanted to make a reminder system for her mother who has a chronic illness and often forgets to take her medication.
